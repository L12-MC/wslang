# Variables and Data Types

## Variable Declaration and Assignment

Variables are created using the assignment operator `=`:

```
>> x = 10
> x = 10.0
```

## Data Types

### Numbers

All numbers are stored as floating-point values:

```
>> integer = 42
> integer = 42.0
>> decimal = 3.14
> decimal = 3.14
>> negative = -7.5
> negative = -7.5
```

### Strings

Strings can be defined with double quotes or single quotes:

```
>> message = "Hello, World!"
> message = "Hello, World!"
>> name = 'Alice'
> name = 'Alice'
```

### Booleans

Boolean values are `true` or `false`:

```
>> isActive = true
> isActive = true
>> isComplete = false
> isComplete = false
```

### Lists

Lists are ordered collections of values:

```
>> myList = [1, 2, 3, 4, 5]
> myList = [1, 2, 3, 4, 5]
>> names = ["Alice", "Bob", "Charlie"]
> names = ["Alice", "Bob", "Charlie"]
```

## Variable Names

Variable names must:
- Start with a letter or underscore
- Contain only letters, numbers, and underscores
- Not be a reserved keyword

Valid examples:
```
>> myVar = 10
>> _private = 20
>> variable123 = 30
>> camelCase = 40
>> snake_case = 50
```

## Printing Variables

Use the `print()` function to display variable values:

```
>> x = 42
> x = 42.0
>> print(x)
42.0
```

## Checking Variable Values

Variables can be used in expressions:

```
>> x = 10
> x = 10.0
>> y = x + 5
> y = 15.0
>> print(y)
15.0
```
