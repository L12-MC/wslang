# Getting Started

## Installation

Make sure you have Dart installed on your system.

## Running the REPL

To start the REPL, run:

```bash
dart main.dart
```

You'll see the prompt:

```
repl v0.2
>>
```

## Your First Commands

### Simple Arithmetic

```
>> 5 + 3
> 8.0
>> 10 * 2
> 20.0
>> (5 + 3) * 2
> 16.0
```

### Variables

```
>> x = 42
> x = 42.0
>> y = x + 8
> y = 50.0
>> print(y)
50.0
```

### Strings

```
>> name = "World"
> name = "World"
>> print(name)
World
```

### Booleans

```
>> flag = true
> flag = true
>> result = 5 > 3
> result = true
```

## Exiting the REPL

Type `exit()` or `quit()` to exit:

```
>> exit()
Exiting...
```
